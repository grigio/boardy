# Boardy
Boardy is a dashboard.

![](http://puu.sh/21AQu)

## Installation
Clone this repo:

    $ git://github.com/jonasschneider/boardy.git
    $ bundle install

Set up hosts to watch:

    $ echo "BOARDYHOSTS=google.com,github.com" > .env

Let [foreman](https://github.com/ddollar/foreman) run it:

    $ foreman start
